Acerca de este sitio web

¡Hola! Bienvenido a ¡Códi-GO!, un lugar dónde los programadores principiantes que hablan espańol pueden aprender a codificar con Scratch, Python, HTML, CSS, JavaScript, y Java. Se recomienda aprender en este orden: Scrach, Python, HTML/CSS/JavaScript, Java, pero, esto no es necesario.


El Formato

Las páginas de cada lenguaje de programación tienen:
- Instrucciones de instalación y cómo probar su código
- Las funciones fundamentales de cada lenguaje de programación (con ejemplos)
- Ejercicios para que pruebe a usar lo que aprendió (con soluciónes)
- Un proyecto guiado, donde podrás usar tus habilidades de programación para crear un proyecto final, como un juego o un sitio web.
- Recursos adicionales para ayudarle a explorar más


¿Preguntas?

Puedes presionar el botón “Foro” en el esquina superior derecha para ver los foros de discusión. Aquí, puedes hacer cualquier pregunta que tengas, y el ¡Codi-GO! equipo o otros usuarios pueden ayudarte.
